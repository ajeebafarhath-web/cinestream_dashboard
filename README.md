# 📖 About the Project

The **CineStream Content Analytics Dashboard** is an interactive data analytics project developed as part of the **Semester 2 OJT Self Project**.

The dashboard analyzes the complete content catalog of the fictional OTT platform **CineStream** and provides business insights through KPIs, interactive charts, filters, and smart status alerts.

# 📂 Dataset Information

**Dataset Name**

> CineStream_Catalog.csv

The dataset contains approximately **520 OTT titles** released between **2015 and 2024** and added to the CineStream platform between **2020 and 2024**.

---

# 📋 Dataset Columns

| Category | Columns |
|----------|----------|
| Basic Information | ContentID, Title, Type, Genre, Language |
| Production | Country, Director, CastCount |
| Release Details | ReleaseYear, AddedDate, RuntimeMinutes, EpisodeCount |
| Ratings | IMDbScore, CriticRating, AgeRating |
| Performance | ViewsMillions, WatchHoursMillions, AvgCompletionPct |
| Financial | ProductionCostCr, RevenueCr, Profit_Cr, ROI_Pct |
| Audience | SubscribersGainedThousands, TrendingStatus, Performance_Band |
